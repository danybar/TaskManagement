Návod k zapnutí projektu na lokálním serveru
==============================================

1. Nainstalovat xampp, docker nebo jinou službu pro nahození virtuálního prostředí
2. Rozjet si na localhost phpmyadmin, kde si následně naimportovat data. (export databáze je přiložen v souborech) 
3. Změnit přístupové údaje k databázi v souboru local.neon.

Přístupové údaje:

Username: Jan
Heslo: test

Uživatel se může také zaregistrovat pod libovolným kódem organizace:

Česká televize: o=m9_UaDayb7
Česká spořitelna: 6*3LB`iz@P_e
Škoda Auto: X&LCAKu_mWoq
